1st Page -- GUI Config
You can select which theme to view and other options that affect the ROM selector menus.
2nd Page -- Emulator Config
You can select which emulator to use as well as if you want to view that platform's titles in the ROM selector at all.
3rd Page -- GBARunner2 Config
Ces paramètres vous permettent de sélectionner la façon dont les jeux sont affichés et de les exécuter plus rapidement ou d'avoir une meilleure compatibilité.
4th Page -- Games & Apps Config
These are the general settings for how games should be run. These are settings regarding launching applications in general. Select whether to use flashcarts or nds-bootstrap, use your LED as read indicators (DSi only), have widescreen enabled (3DS only) and more.
5th Page -- Misc. Config
The options in this page toggles which language to use, whether to automatically boot the last played game, whether to show the splash screens on startup, amongst others.
Paramètres d'Unlaunch
Ces paramètres vous permettent de changer le fonctionnement d'Unlaunch, ils fonctionnent en patchant l'installateur Unlaunch, donc vous devrez le réinstaller pour que ces paramètres soient pris en compte.
