BIOS File
GBARunner2 requires a dump of the GBA BIOS titled bios.bin. This file could be located either at the root of the SD card, /gba folder or /_gba folder.
Make sure your GBA BIOS dump matches with the following CRC32 hash: 81977335.
You can check this by visiting the following website: hash.online-convert.com/crc32-generator
