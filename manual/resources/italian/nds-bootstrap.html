Informazioni

nds-bootstrap is a homebrew application used by TWiLight Menu++ to load cartridge dumps and DS-mode homebrew from the Nintendo DSi/3DS SD card.

There is also a version for flashcarts titled B4DS, intended for homebrew-only flashcarts and flashcards with low compatibility.

Compatibilità

To see if a game is compatible with nds-bootstrap, check the compatibility list:
cutt.ly/nds-compatibility

Per avere una migliore compatibilità con B4DS, si consiglia di ottenere un DS Memory Expansion Pak.
