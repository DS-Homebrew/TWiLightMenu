File BIOS
GBARunner2 può utilizzare un dump del BIOS ufficiale del GBA per migliorare la compatibilità. Il file deve essere chiamato bios.bin e posizionato nella Scheda SD,insd:/gba o insd:/_gba.
Assicurati che il tuo dump Bios del GBA corrisponda a questo hash CR32: 81977335.
Puoi controllarlo visitando il seguente sito web: simplycalc.com/crc32-file.php
